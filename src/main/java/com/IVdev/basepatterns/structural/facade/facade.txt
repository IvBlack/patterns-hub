Шаблон: фасад.
****************
Суть:
предоставить унифицированный интерфейс взамен нескольких.
****************

Для чего:
для опеределения интерфейса высокого уровня, который упрощает использование подсистемы.
****************

Пример:
- необходимость ослабления связанности подсистемы с клиентами.
- изолирование клиентов от компонентов подсистемы, упрощая работу с ней.
