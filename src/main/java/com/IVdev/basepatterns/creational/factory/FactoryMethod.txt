Шаблон: фабрика (factory method)
****************
Суть:

Создание интерфейса, который создает объект.
При этом выбор того, экземпляр какого класса будет создан, остается за классами,
что реализуют данный интерфейс.
****************

Для чего:
Делегирование создания объекта другому клссу.
****************

Пример:
- заранее неизвестно, экземпляры какого класса нужно будет создавать;
- класс спроектирован так, что создаваемые им объекты имеют свойства опр.класса;
